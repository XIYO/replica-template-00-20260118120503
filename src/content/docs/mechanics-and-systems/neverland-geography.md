## 1. 개요

네버랜드 지리 정보 시스템(Neverland Geographic Information System, NGIS)은 섬 내의 복잡한 지형과 비선형적인 시공간 특성을 데이터화하여 관리하기 위한 시스템입니다. 본 문서는 시스템의 좌표계 기준과 주요 거점(POI)의 지리적 속성을 기술합니다.

## 2. 좌표 체계 및 데이터 구조

네버랜드는 고정된 위도와 경도 대신, **'오른쪽에서 두 번째 별'**을 원점(0,0,0)으로 하는 상대 좌표계를 사용합니다. 

### 2.1 데이터 필드 정의

- `id`: 거점 고유 식별자
- `coordinates`: [X, Y, Z] 축 데이터
- `magic_density`: 마법 에너지 밀도 (0.0 ~ 1.0)
- `time_distortion`: 시간 왜곡 계수

## 3. 주요 거점별 지리적 특성

### 3.1 머메이드 라군 (Mermaid Lagoon)

해안가에 위치한 산호초 지대로, 수중과 지상의 경계가 모호한 복합 지형입니다.

- **특징**: 수심이 깊어질수록 마법 밀도가 급격히 상승하며, 전파 방해 현상이 발생함.
- **좌표**: `[150.22, -405.8, -12.5]`
- **주요 위험 요소**: 사이렌 주파수 간섭, 급격한 조류 변화.

### 3.2 인디언 마을 (Tiger Lily's Territory)

섬의 동북쪽 고지대에 위치한 전략적 요충지입니다.

- **특징**: 울창한 삼림으로 둘러싸여 있어 천연의 요새 역할을 수행하며, 토양 내 영적 에너지가 높음.
- **좌표**: `[842.15, 312.44, 158.0]`
- **주요 자원**: 고농축 마법 약초, 고대 토템 잔해.

### 3.3 해골 바위 (Skull Rock)

간조 시에만 접근 가능한 고립된 석회암 지형입니다.

- **특징**: 내부 동굴의 반향 현상이 심하며, 시간에 따라 입구의 좌표가 미세하게 변동함.
- **좌표**: `[-920.0, -110.5, 5.0]` (가변적)

## 4. 실용적인 API 데이터 예제

개발자는 다음 JSON 구조를 참고하여 NGIS 서버에서 위치 데이터를 쿼리할 수 있습니다.

```json
{
  "location_id": "LOC-001",
  "name": "Mermaid Lagoon",
  "metadata": {
    "terrain": "Coastal/Reef",
    "accessibility": "Sea only",
    "coordinates": {
      "x": 150.22,
      "y": -405.8,
      "z": -12.5
    },
    "magic_stats": {
      "density": 0.85,
      "stability": "Low"
    }
  }
}
```

## 5. 관리 및 주의사항

1. **시공간 변동**: 네버랜드의 지형은 거주자의 '믿음' 수치에 따라 실시간으로 렌더링 오차가 발생할 수 있습니다.
2. **데이터 동기화**: 매일 자정(네버랜드 시간 기준)마다 지리 데이터의 정합성 검사를 권장합니다.
3. **오차 범위**: NGIS 좌표는 최대 ±5m의 마법적 오차 범위를 가질 수 있음을 인지해야 합니다.
